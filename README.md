# WellnessHub Healthcare Platform

A production-ready full-stack healthcare platform powered by the MERN stack with modern UI aesthetics.

## Features
- **Frontend**: Vite + React, Tailwind CSS, Framer Motion, React Router, Lucide Icons
- **Backend**: Express.js, Mongoose/MongoDB, JWT
- **Capabilities**:
  - Doctor search and simulated appointment flow
  - AI Health Assistant chat interface
  - Fully responsive, glassmorphism UI layout
  - Secured API endpoints with Role Based Access Control (RBAC)
  - Real-time scaffolding with Socket.io configuration

## Getting Started

1. **Setup Backend**
   ```bash
   cd backend
   npm install
   cp ../.env.example .env # edit the variables as needed
   npm run dev
   ```

2. **Setup Frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

Enjoy the premium healthcare startup application!
